# [LT-06] 라오트러스트(LaoTrust) 서비스 스토리보드 v1.0

| 항목 | 내용 |
|------|------|
| **문서 번호** | LT-06 |
| **작성일** | 2026-02-20 |
| **작성자** | 수사본부 에이스 첫째 (Gemini) |
| **대상** | 개발 요원 막내 (Claude/Cursor) |

---

## 🧭 1. [사용자 경로 A] 전문가 견적 요청 (70% - 숨고 스타일)

기술이 필요한 유저가 전문가를 불러 매칭되는 핵심 동선입니다.

| 순서 | 화면/액션 | 설명 |
|------|-----------|------|
| 1 | **[홈 화면]** | 메인 대시보드에서 '에어컨 수리' 아이콘 클릭 |
| 2 | **[질문지 Step 1]** | 증상 선택 (예: 찬바람 안 나옴) → '다음' 클릭 |
| 3 | **[질문지 Step 2]** | 위치 및 희망 시간 선택 → '다음' 클릭 |
| 4 | **[사진/상세]** | 현장 사진 업로드 및 추가 요청사항 입력 → '제출' 클릭 |
| 5 | **[완료 팝업]** | "요청이 완료되었습니다! 전문가의 견적을 기다려주세요." 노출 |
| 6 | **[매칭 리스트]** | 하단 '채팅' 탭으로 이동하여 도착한 견적서들 확인 |

---

## 🧭 2. [사용자 경로 B] 급구 알바 지원 (30% - 알바몬 스타일)

일자리가 필요한 유저가 주변 공고를 보고 즉시 지원하는 동선입니다.

| 순서 | 화면/액션 | 설명 |
|------|-----------|------|
| 1 | **[홈 화면]** | 하단 'Jobs' 탭 클릭 또는 메인 리스트의 알바 카드 클릭 |
| 2 | **[알바 리스트]** | 현재 위치 기반의 구인 공고 목록 확인. (필요 시 '지도보기' 전환) |
| 3 | **[공고 상세]** | 관심 있는 공고 클릭하여 상세 업무 및 급여 확인 |
| 4 | **[지원하기]** | '간편 지원' 버튼 클릭 → 내 프로필 정보 전송 승인 |
| 5 | **[채팅방 생성]** | 고용주와 즉시 1:1 채팅방 개설 및 대화 시작 |

---

## 🧭 3. [비즈니스 경로 C] 전문가 신뢰 인증 (결제 동선)

기술자가 플랫폼 내에서 신뢰를 얻고 수익을 창출하는 동선입니다.

| 순서 | 화면/액션 | 설명 |
|------|-----------|------|
| 1 | **[내 정보]** | 하단 'Profile' 탭 클릭 |
| 2 | **[인증 신청]** | 'Verified Badge(인증 배지) 신청하기' 버튼 클릭 |
| 3 | **[결제 안내]** | 인증의 혜택(상단 노출 등) 및 비용(4.5 USD) 안내 확인 |
| 4 | **[ID 업로드]** | 여권 또는 라오스 ID 카드 촬영 및 업로드 |
| 5 | **[결제 진행]** | BCEL OnePay(QR 결제) API 호출 및 결제 완료 |
| 6 | **[대기/승인]** | "관리자 검토 중입니다." 안내 → 승인 시 이름 옆에 파란 배지 생성 |

---

## 🛠️ 막내(Claude) 요원을 위한 기술 구현 지침 (Logic Rules)

| 규칙 | 내용 |
|------|------|
| **화면 전환 애니메이션** | 모든 화면 전환은 부드러운 **슬라이드(Slide)** 효과를 적용할 것 |
| **상태 보존** | 사용자가 질문지 작성 중 앱을 껐다 켜도 입력한 내용이 유지되어야 함. **(Local State Persistence)** |
| **뒤로가기 규칙** | 질문지 단계에서는 상단 '←' 버튼 클릭 시 **바로 전 단계 질문**으로 이동할 것 |

---

> 본 스토리보드는 형님의 최종 컨펌 후 막내에게 **'개발 개시'** 명령과 함께 전달됩니다.
