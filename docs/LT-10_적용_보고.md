# LT-10 실전 코딩 정밀 지시서 적용 보고

## 1. Firebase 통신망 및 보안 체계

| 항목 | 적용 내용 |
|------|-----------|
| **[Auth]** | `lib/core/phone_auth_service.dart`: 라오스 현지 전화번호(+856) 기반 Phone Auth 초안. `normalizeLaosPhone`, `sendPhoneVerificationCodeLaos`, `signInWithPhoneCodeLaos`. |
| **[Firestore]** | `lib/data/firestore_schema.dart` (LT-10 반영): **users** — `is_verified`, `user_type`(expert/general) 필수. **requests** — 3단계 질문 데이터(symptoms, location, wishedTime, photoUrl, extraNote). **jobs** — `location_geo`(GeoPoint) 추가, 위치 기반 저장 구조. |

## 2. Indigo Blue 테마 엔진 및 UI 뼈대

| 항목 | 적용 내용 |
|------|-----------|
| **[Theme]** | `lib/theme/app_theme.dart`: **#3F51B5** 전역 고정. light/dark 모두 primary = indigoBlue(#3F51B5). |
| **[Navigation]** | `lib/features/main_tab/main_tab_screen.dart`: 하단 탭 바(Home, Jobs, Chat, Profile) 구현. **IndexedStack**으로 4개 화면 유지 → 탭 전환 시 **상태 보존(State Persistence)**. 앱 초기 라우트 = MainTabScreen. |
